# Seva Sahayak - AI-Powered Form Filling Assistant

An AI-powered document extraction and form-filling assistant designed for Indian citizen services.

## Features

- **Multi-document Support**: Extract data from Aadhaar, PAN, Voter ID, Passport, and Driving License
- **AI-Powered OCR**: Intelligent document parsing with high accuracy
- **Multi-language Support**: Available in English, Hindi, and Marathi
- **Voice Input**: Voice-enabled data entry for accessibility
- **Smart Form Filling**: Auto-populate government forms with extracted data

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- Supabase (Backend)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
